# Geometry Generation Server

FastAPI service for image-conditioned 3D geometry generation. Supported
backends are SAM3D and Hunyuan3D. Successful responses include the visual GLB
plus mandatory collision assets generated by `postprocess_server`.

## Run

```bash
uv run python -m assetserver.geometry_generation_server.standalone_server \
  --host 0.0.0.0 \
  --port 7000 \
  --backend sam3d
```

```bash
uv run python -m assetserver.geometry_generation_server.standalone_server \
  --host 0.0.0.0 \
  --port 7002 \
  --backend hunyuan3d
```

## API

- `GET /health`
- `POST /generate_geometries`
- `GET /assets/{asset_id}`

`POST /generate_geometries` accepts a list of requests containing `image_path`,
`output_dir`, and `prompt`. It returns an NDJSON stream. Successful rows include
`geometry_path`, `asset_id`, `download_url`, and `collision`.
